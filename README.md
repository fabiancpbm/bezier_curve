# bezier_curve
Curva de Bezier feita para a disciplina de Computação Gráfica.

- Para executar, basta executar o Play
- A constante DELTA_P1_P2 modifica a distância entre os pontos P1 e P2.
- A constante DIMENSION indica a dimensão da curva de bezier. O programa está preparado para trabalhar por enquanto apenas com 3 dimensões, ou seja, 4 pontos de referência.
